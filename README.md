# Binary Distillation Column Design using McCabe–Thiele Method

This project generates a graphical distillation column design in Excel using the McCabe–Thiele method by assuming constant relative volatality

## Features
- Equilibrium curve generation
- Rectifying and stripping operating lines
- Feed line implementation
- Graphical stage determination
- Reflux ratio sensitivity analysis

## Tools Used
- Microsoft Excel

## Result
For a reflux ratio of 2, approximately 7 theoretical stages are required to achieve the desired separation.
